# Motion Brief — by @AmirMushich & OpenArt

Free Claude Project that turns any brand image
into a production-ready Seedance 2.0 video prompt —
and shows you how to sell the result.

## How to set up

**Option A — Claude Project (recommended)**
1. Create a new Claude Project at claude.ai
2. Upload all files from this repo to the Project
3. Paste the contents of `INSTRUCTION.md`
   into Project Instructions
4. Start a new chat — upload any brand image

**Option B — Web App**
→ [Open Motion Brief Web App](amirmushich.link/MoBrief)

## Files
- `INSTRUCTION.md` — system prompt for Claude Project
- `pricing.md` — pricing framework (market benchmarks)
- `channels-buyers.md` — where and how to sell
