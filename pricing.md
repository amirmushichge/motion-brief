## US Base Rates (Market Benchmark)

| Deliverable | Range (USD) | Notes |
|---|---|---|
| Animated GIF (3–5s, brand-derived) | $150–400 | Batch / derivative asset |
| Animated GIF (custom concept) | $400–750 | Includes design + refinement |
| Email hero / banner animation (8–15s) | $500–1,200 | Smaller clients |
| Email hero / banner animation (8–15s) | $1,200–2,000+ | Bigger brands, multiple sizes |
| Short brand loop (5–15s, social/web) | $800–2,500 | Custom motion language |
| Logo reveal / brand intro | $750–2,000 | High-end, custom |
| Full brand motion package | $3,000–5,000+ | Logo + loops + launch assets |

**Day rate anchor:** $500–900/day (mid-to-senior, custom work)
**Floor:** Never go below $350–400 for any custom animated asset.

## Level-Based Positioning

### Level 1 — Deployment-Ready Assets
Solo creator, no existing client base.
Selling per-channel asset packages.

```
Email GIF package:        $300–600
Paid ads animated set:    $400–800
E-commerce loop:          $500–900
Pitch deck opener:        $400–700
```

### Level 2 — Brand System Sale
Designer with client base.
Motion assets included in brandbook as standard.

```
Brandbook + motion kit:   same fee, higher perceived value
Standalone motion add-on: $800–2,000 per project
```
*Do not price motion as a "bonus." It's standard.*

### Level 3 — Agency Partnership
White-label subcontractor model.

```
Day rate (internal):      $500–800/day
Per-project (30–90s):     $1,500–4,000
Monthly retainer:         $2,000–5,000/month (part-time)
Monthly retainer:         $4,000–7,000/month (dedicated senior)
```

*Agency marks up 2–3× to end client. Price accordingly.*

---

## Pricing Logic — One Rule

**Frame assets as systems, not one-offs.**

One animated email hero + static fallbacks = 2–3 days work
→ Justify $900–1,500 easily.

Three branded loops for social (multiple ratios)
→ $1,800–3,500 is defensible.

You are not selling "a GIF."
You are selling a deployment-ready asset for a revenue channel.
